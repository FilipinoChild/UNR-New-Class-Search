Connect to database:
1.) Go to neon dashboard
2.) Click Connect
3.) Copy Connection String snippet
4.) For windows change single quotes '' to double quotes ""
    For Mac leave as is
5.) Paste into terminal
6.) Done! connected to database

Once Connected test out
\l  to list databases
\h  for commands
\dn view schemas
\q  quit

Work with a certain Schema: 
1) SET search_path TO sales, public;
2) SHOW search_path;
